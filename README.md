Aplicacion creada para guardar el progreso de una serie cruzada, guarda las opciones marcadas y colores en funcion de la serie para diferenciar agilmente, tambien cambia de color depende si fue visto o no, y siempre al iniciar te lleva al proximo capito a revisar, hay opciones para cambiar la lista por si hay algun cambio y ademas un boton para guardar el utlimo capitulo visto, tambien esta opcion permite cargar el ultimo capitulo visto marcando todos los anteriores como vistos tambien, hay botones para subir al inicio o ir al que se esta viendo
